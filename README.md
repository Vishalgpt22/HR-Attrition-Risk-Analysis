# HR Attrition Risk Analysis

## Project Overview
This project analyzes employee attrition risk using SQL,Tableau and Python.

## Tools Used
- Excel
- MySQL
- Tableau
- Python 
- GitHub

## Dashboards Created
1. Employee Attrition Executive Overview Dashboard
2. Hr Retention Risk Dashboard

## Key Insights
- High risk employees identified using promotion delay logic
- Department-wise attrition trends
- Job role risk concentration
- Promotion gap impact on attrition

## Repository Structure
data/ – Raw datasets  
sql/ – SQL queries and views  
dashboards/ – Tableau dashboards  
python/ – Data preprocessing scripts
